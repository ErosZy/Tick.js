Tick.js
=======

a lightweight animation library for javascript


##Usage

```
	/*
 	params:{
 				attr:val,
 				....,
 				onComplete : function,
 				onCompleteParam : []
 			}
	 */

	Tick.to(target,params,'easeInCubic',1000);

	Tick.from(target,params,'easeInCubic',1000);

	Tick.fromTo(target,tParams,fParams,'easeInCubic',1000);

```
